# 開啟應用程式 (Terminal)

* [設定片段](https://github.com/samwhelp/note-about-i3wm/blob/gh-pages/_demo/config/i3wm-config/main/config/i3/share/gen/i3wm-gen-rc/Section/Subject/Application/Keybind/Terminal.conf)


| 按鍵組合           | 功能         | 執行指令         |
| ----------------- | ------------- | ---------------- |
| `Alt + Enter`     | 開啟 Terminal | `sakura`         |
| `Alt + Shift + a` | 開啟 Terminal | `sakura`         |
| `Alt + Ctrl + a`  | 開啟 Terminal | `xfce4-terminal` |
| `Alt + Shift + t` | 開啟 Terminal | `xterm`          |
| `Alt + Ctrl + t`  | 開啟 Terminal | `urxvt`          |
